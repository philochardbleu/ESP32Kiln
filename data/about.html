ESP32Kiln Controller

ESP32Kiln is an ESP32 board based kiln controller designed for precision temperature control in high-temperature applications.

Applications
  • Ceramic firing
  • Glass fusing and melting
  • Metal melting (soft metals)
  • Temperature range: up to 1350°C
Features
  • PID temperature control
  • Web interface & LCD display
  • Program scheduling
  • Data logging & monitoring

Author: Adrian Siemieniak, modded by Alejandro Mora
